# Perfil Altimétrico Ferroviario Interactivo

Este script de Python genera un perfil altimétrico interactivo de un tramo ferroviario a partir de un archivo CSV que contiene las estaciones y sus coordenadas. Utiliza la API de Open-Elevation para obtener los datos de elevación y la biblioteca Plotly para crear un gráfico interactivo que se guarda como un archivo HTML.
